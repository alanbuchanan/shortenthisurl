Alanbuchanan's URL Shortener

This is a small service for returning an alias to an existing URL. It can be used as a URL shortener.

Go ahead and append this url with /new/ followed by a URL of your choice, and you'll receive an alias of the URL.

Example input:

https://shortenthisurl.herokuapp.com/new/github.com

Example output

{ shortUrl: "https://shortenthisurl.herokuapp.com/iH7Z", originalUrl: "http://github.com" }
